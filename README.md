# Redstone Terrain Designer

マインクラフト統合版用の **Redstone 自動整地回路設計ツール**です。

## 🎯 機能

- ✅ **3D ビュー** - 立体的に回路設計を確認
- ✅ **俯瞰図・側面図** - 複数の角度から設計を検討
- ✅ **必要アイテム計算** - 自動で必要なピストン・リピーター・レッドストーンを計算
- ✅ **組み立て手順** - ステップバイステップの説明
- ✅ **エクスポート機能** - 画像で保存、テキストでコピー可能
- ✅ **Realms 安全動作** - コマンド不要、Redstone 回路のみ使用

## 📋 使い方

### 1. 座標入力
```
開始座標 (X1, Z1)
終了座標 (X2, Z2)
```

### 2. 掘り下げ設定
```
掘り下げる深さ (Y軸)
```

### 3. 積み上げ設定
```
積み上げる高さ
ブロック種類（土、石、砂など）
```

### 4. 回路スピード選択
```
遅い (安定) → 速い → 超高速
```

### 5. 「設計図を生成」をクリック
- 3D ビューで視認化
- 必要アイテムリスト表示
- 組み立て手順が表示されます

## 📐 設計図の見方

### 3D ビュー
- 🟢 **緑色** = 積み上げエリア（ブロック積み上げ）
- 🔴 **赤色** = 掘り下げエリア（土を掘り下げ）

### 俯瞰図（上か��見た図）
- 🔵 **青** = ピストン配置
- 🟠 **黄** = リピーター配置
- 🔴 **赤** = レッドストーン配線

### 側面図（横から見た図）
- ピストンとリピーターの垂直配置
- 掘り下げと積み上げの高さ関係

## ⚙️ 必要な Redstone コンポーネント

| アイテム | 用途 | 個数 |
|---------|------|------|
| ピストン | ブロック掘り下げ・積み上げ | 領域幅に応じて |
| リピーター | 信号遅延・タイミング制御 | 領域幅に応じて |
| レッドストーン | 信号配線 | 領域幅に応じて |
| レバー | 起動スイッチ | 1個 |
| チェスト / ホッパー | ブロック供給 | 2個 |

## 🚀 クイックスタート

### 例1: 32×32 の平坦地作成
1. X1=0, Z1=0, X2=32, Z2=32
2. 掘り下げ深さ = 5
3. 積み上げ高さ = 3
4. ブロック種類 = 土
5. 生成 → 設計図表示

### 例2: 建築予定地準備
1. 建築したい座標範囲を入力
2. 回路スピードを「通常」に設定
3. 必要アイテムをリストから確認
4. 手順に従って Redstone 回路を構築

## ✅ Realms での安全性

このツールは以下の理由で **Realms で安全に使用** できます：

- ✓ コマンド（`/fill` など）を使用していない
- ✓ 100% Redstone 回路（ゲーム内機構）のみ
- ✓ 利用規約違反なし
- ✓ マルチプレイサーバー対応
- ✓ アカウント制限の対象にならない

## 📝 使用上の注意

1. **大規模範囲の場合**
   - 一度に大きすぎる範囲を処理しないこと
   - 小分けにして複数回実行することを推奨

2. **テスト実行**
   - 実際に回路を構築する前に、小さな範囲でテストしてください
   - 十分なセーブポイントを作成してから実行してください

3. **パフォーマンス**
   - Realms サーバーの負荷を考慮してください
   - 複数プレイヤーがいる場合は、深夜等に実行することを推奨

4. **ブロック供給**
   - 大量のブロックが必要な場合、供給装置を先に準備してください

## 🎓 Redstone 回路の基礎

### ピストン
- ブロックを 1 マス押し出す（または引き込む）
- 信号を受けると動作

### リピーター
- レッドストーン信号を中継
- 遅延を設定可能（1～4 ティック）
- 信号強度を回復

### レッドストーン
- 電気信号のようなもの
- 15 ブロック先まで信号が伝播

## 💡 トラブルシューティング

### 「設計図が表示されない」
→ 座標が正しく入力されているか確認してください

### 「必要アイテムが多すぎる」
→ 掘り下げ深さ・積み上げ高さを減らしてみてください

### 「領域が小さい」
→ X1, Z1, X2, Z2 の座標差をもっと大きくしてください
